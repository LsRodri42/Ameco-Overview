# Organização de Mídia

Esta pasta existe para concentrar os materiais visuais do projeto AMECO no repositório de overview.

## Estrutura

- `assets/screenshots/`
- `assets/video/`

## Vídeo de Demonstração

Coloque o vídeo principal do aplicativo em:

- `assets/video/`

Nome recomendado:

- `ameco-demo-v1.mp4`

Se houver outras versões:

- `ameco-demo-v2.mp4`
- `ameco-demo-v3.mp4`

## Convenção recomendada

- usar nomes em minúsculo
- separar palavras com hífen
- evitar espaços e acentos
- preferir `mp4`

## Observação importante

Se o vídeo for muito grande, o ideal é não mantê-lo versionado diretamente no Git por muito tempo.

Melhor prática:

1. manter uma cópia local para organização
2. publicar a demonstração por link externo ou release, se necessário
3. deixar o `README` apontando para a versão oficial do vídeo

## Próximo passo

Depois que o vídeo for adicionado:

1. decidir se ele ficará no repositório, release ou plataforma externa
2. inserir o link no `README`
3. fazer commit e publicar
