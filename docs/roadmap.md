# Roadmap do AMECO

## Fase Atual

### Base Funcional

- app mobile funcional
- backend funcional
- autenticação integrada
- módulos de energia e água em desenvolvimento contínuo
- IA conversacional em nível inicial

## Próxima Fase

### Organização e Padronização

- criar documentação central
- revisar estrutura dos repositórios
- padronizar nomes de arquivos e módulos
- melhorar scripts de execução
- documentar arquitetura e setup

## Evolução da IA

### Nível 1 - Conversacional

- explicar dados
- responder perguntas
- sugerir economia

### Nível 2 - Analítica

- analisar histórico
- comparar períodos
- apontar desperdícios

### Nível 3 - Preditiva

- prever consumo
- gerar alertas
- sugerir ações personalizadas

## Melhorias Técnicas

- migração do backend para TypeScript
- testes automatizados
- contratos de API documentados
- separação mais rígida entre controller, service e integração

## Melhorias de Produto

- gráficos mais claros
- histórico consolidado
- leitura assistida de contas
- trilhas de educação sustentável
