# Organização de Screenshots

Esta pasta existe para concentrar as principais telas do aplicativo no repositório de overview.

## Estrutura

- `assets/screenshots/auth/`
- `assets/screenshots/dashboard/`
- `assets/screenshots/scanner/`
- `assets/screenshots/energia/`
- `assets/screenshots/agua/`
- `assets/screenshots/ia/`
- `assets/screenshots/perfil/`
- `assets/screenshots/sustentabilidade/`

## Como adicionar

Coloque cada print na pasta correspondente ao fluxo da tela.

Exemplos:

- `assets/screenshots/auth/login.png`
- `assets/screenshots/auth/cadastro.png`
- `assets/screenshots/dashboard/home.png`
- `assets/screenshots/scanner/captura-conta.png`
- `assets/screenshots/scanner/revisao-dados.png`
- `assets/screenshots/energia/calculo-energia.png`
- `assets/screenshots/agua/calculo-agua.png`
- `assets/screenshots/ia/chat-ameco.png`
- `assets/screenshots/perfil/perfil-usuario.png`
- `assets/screenshots/sustentabilidade/dicas-sustentaveis.png`

## Convenção recomendada

- usar nomes em minúsculo
- separar palavras com hífen
- evitar espaços e acentos
- preferir `png` ou `jpg`

## Próximo passo

Depois que as imagens forem adicionadas, o próximo ajuste será:

1. selecionar as telas mais importantes
2. inserir as imagens no `README`
3. fazer commit e publicar no GitHub
